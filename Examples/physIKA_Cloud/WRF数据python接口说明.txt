win10上操作过程：
    在.\xml文件夹下运行python tcdsm_run.py 上传的xml路径 图像所在文件夹 时间戳文件夹 生成的模拟数据存放的文件夹
如：
python tcdsm_run.py 
D:\Code\PhysIKA_Cloud\Python\TCDSM\xml\2021\upload_config_file.xml 
D:\Code\PhysIKA_Cloud\Python\TCDSM\data （文件名要写在上面的xml中）
D:\Code\PhysIKA_Cloud\Python\TCDSM\xml\2021 
D:\Code\PhysIKA_Cloud\Python\TCDSM\xml\2021\sim_data


新的xml文件路径会输出到控制台上

会在当前目录下生成logs文件夹
